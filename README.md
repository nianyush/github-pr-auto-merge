# github-pr-auto-merge

## Description

This action is a chrome extension that will automatically check if a PR is ready to be merged. It will first check if the PR has been updated with the latest changes from the base branch. If it has, it will then check if the merge button is enabled. If it is, it will automatically merge the PR.

## Usage

Clone the repository and enable developer mode in chrome extensions. Then click on load unpacked and select the repository folder. The extension will be added to your chrome browser.
